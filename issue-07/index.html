明星项目

1、LingDong-/wenyan-lang

近日，一位 卡内基梅隆大学（CMU）的大四学生开发了基于 文言文 的编程语言，高中语文三大怕的文言文终于找上程序员了。这一项目并不是简单的将程序中的英文字符换成了中文，而是利用 NLP 的一些技术，将文言文程序语法转换到 JavaScript 或 Python 运行，有一定的技术难度。目前已收割 10K + Star。
